# NeoSapien Desktop App

## Setup

Install dependencies:

```bash
npm install
```

Create `.env` file from `.env.example`:

```bash
cp .env.example .env
```

Configure environment variables in `.env` file.

## Development

Start development server:

```bash
npm start
```

Build for production:

```bash
npm build
```

Format code:

```bash
npm run format
```
